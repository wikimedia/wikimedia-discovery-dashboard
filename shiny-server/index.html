Discovery Dashboards

What is this?

This is a collection of dashboards and tools for exploring Wikimedia search data. It contains everything from API usage to direct user interactions, and will only grow over time. The idea is that this will provide data for internal use and external use, to see how well we're doing.

What do I do if I have bug reports, or ideas for future projects?

We're always happy to hear new ideas!

Who built it and how?

This project was built by Oliver Keyes on top of RStudio's Shiny platform.
